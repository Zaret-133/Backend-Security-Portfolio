# API Security Labs

## Overview
This repository contains hands-on API Security and Backend Security projects focused on real-world vulnerabilities, secure development practices, and DevSecOps concepts.

The goal of this portfolio is to document practical security testing, vulnerability analysis, remediation strategies, and automation techniques used to secure modern applications and APIs.

## Focus Areas
- OWASP API Top 10
- Authentication & Authorization Testing
- JWT & Token Security
- API Vulnerability Assessment
- Secure CI/CD Pipelines
- DevSecOps
- Python Security Automation
- Docker Security Labs
- Backend Security Testing

## Tools & Technologies

- Jenkins

- Burp Suite
- OWASP ZAP
